Добро пожаловать на официальный сайт проекта **«Стиль Осетрова»**.  

Здесь опубликованы правовые документы, регулирующие использование уникального художественного стиля Дмитрия Осетрова — автора раскадровок к фильмам *«Девятая рота»*, *«Сталинград»*, *«Монгол»*, *«Есенин»* и другим.  

Проект обеспечивает легальную возможность обучения искусственного интеллекта «руке Осетрова» и создания новых произведений в его творческом ключе.  

Процедура лицензирования осуществляется через ЭДО, а все версии документов и публичные акцепты публикуются в репозитории GitHub для обеспечения прозрачности и верифицируемости.

Подробнее об условиях и правовой модели — [ознакомьтесь с Офертой](/oferta.html).
